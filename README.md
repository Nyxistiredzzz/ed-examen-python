# recu_entorns

## T2.1 – Reanomenat de variables amb VS Code

Les variables originals eren poc descriptives. Es van renomenar utilitzant la funcionalitat **Rename Symbol/Cambiar el nombre del símbolo** de VS Code:

1. Fes clic dret sobre la variable que vols canviar.
2. Selecciona **Rename Symbol** (o prem `F2`).
3. Escriu el nou nom i prem `Enter`. VS Code actualitza automàticament totes les referències al projecte.

| Nom original | Nom nou          | Fitxer         |
|--------------|------------------|----------------|
| `x`          | `nom_fitxer`     | `main.py`      |
| `llista1`    | `bucle_actiu`    | `main.py`      |
| `b`          | `paraula_usuari` | `main.py`      |
| `res`        | `resultat`       | `main.py`      |
| `funcio1`    | `escriure_linia` | ambdós fitxers |
| `a`          | `nom_fitxer`     | `utilitats.py` |
| `b`          | `contingut`      | `utilitats.py` |
